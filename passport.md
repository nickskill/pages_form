## 2. Описание предметной области

Краткое описание предметной области (5–7 предложений):  

Предметная область "Список пользователей системы" предназначена для хранения списка пользователей. Система позволяет регистрировать новых пользователей через форму. В системе предусмотрена возможность редактирование списка пользователей. Оператов системы может просматривать список зарегистрированных в системе пользовтелей и редактировать его.
